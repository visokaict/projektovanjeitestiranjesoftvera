# Objašnjenje

Ovaj solution se sastoji od dva projekta: jedan je glavni projekat, a drugi je testni projekat. Glavni projekat implementira jednostavnu funkcionalnost aplikacije, dok testni projekat sadrži jedinične (unit) testove za proveru ispravnosti koda glavnog projekta.

Ovaj solution se koristi kao primer koji se daje studentima u okviru AzureDevOps demonstaracije. Cilj je da studenti nauče kako da koriste Azure DevOps za upravljanje projektima, verzionisanje koda, i automatizaciju procesa testiranja i isporuke softvera.
